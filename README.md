UML Diagram Maker
##A UML Diagram Maker built using Java and JavaFX, designed to simplify the creation of UML diagrams. 
##This tool provides an intuitive interface for visualizing and organizing software architecture, 
##developed with compatibility in mind using the latest Java SDK.

Requirements
#To run this UML Diagram Maker, ensure you have the following:
##Java SDK version 23 or later
##JavaFX Libraries
##IntelliJ IDEA or any other preferred IDE
